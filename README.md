# Advanced Algorithms Coursework

This repository contains my coursework for the **Advanced Algorithms (UFCFYR-15-2)** module.

The project is based on implementing different algorithms in Python to solve real-world problems such as data processing, searching, and performance optimisation.

---

## 📌 What is included

This coursework is divided into four main tasks:

### 1. Degree Classification System

* Reads student data from CSV files
* Calculates final degree based on weighted averages
* Outputs results into a file

---

### 2. Password Generator

* Generates valid passwords based on given rules
* Uses combinations of letters, numbers, and symbols
* Saves all generated passwords into a file

---

### 3. Train Route Search

* Finds the cheapest or fastest route between stations
* Uses graph-based logic
* Displays the route and total cost/time

---

### 4. Parallel Programming

* Improves performance of an existing program
* Converts serial processing into parallel execution
* Uses multiple CPU cores

---
